ELECTRICAL NODE DESIGNER - EXCEL ADD-IN
========================================
Version 1.0 | Goodly Insights

REQUIREMENTS
------------
- Windows 10 or later
- Microsoft Excel 2016 or later
- Node.js LTS (free) from https://nodejs.org

INSTALLATION (First time only)
-------------------------------
1. Install Node.js from https://nodejs.org
   Choose the LTS version and install with default settings.

2. Double-click "Install-to-Excel.bat"
   Click YES when Windows asks for Administrator permission.
   Wait for all 7 steps to complete.

DAILY USE
---------
1. Double-click "Start-AddIn.bat"
   Keep this window open while using Excel.

2. Open Excel
   The add-in appears in your Home tab ribbon automatically.
   OR go to: Insert > Add-ins > My Add-ins > Shared Folder tab
   Select "Electrical Node Designer" and click Add.

STOPPING
--------
- Double-click "Stop-AddIn.bat"
- Or close the Start-AddIn.bat window

UNINSTALLING
------------
- Double-click "Uninstall.bat"

TROUBLESHOOTING
---------------
Q: Add-in shows blank or certificate error
A: Run Install-to-Excel.bat again as Administrator

Q: Add-in not showing in Shared Folder tab
A: Close Excel, run Install-to-Excel.bat, reopen Excel

Q: Server won't start - Node.js error
A: Install Node.js from nodejs.org (LTS version)

Q: Works on my machine but not colleague's machine
A: They need to run Install-to-Excel.bat on their machine too
