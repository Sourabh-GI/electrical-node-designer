ELECTRICAL NODE DESIGNER - EXCEL ADD-IN
========================================

INSTALLATION INSTRUCTIONS
--------------------------

METHOD 1 - Automatic (Recommended):
1. Double-click "Install-ElectricalNodeDesigner.bat"
2. If Windows asks "Do you want to allow this app to make changes?" click Yes
3. Follow the on-screen instructions
4. Open Excel -> Insert -> Add-ins -> Shared Folder -> Electrical Node Designer -> Add

METHOD 2 - Manual:
1. Open Excel
2. Go to Insert -> Add-ins -> My Add-ins
3. Click "Upload My Add-in"
4. Browse to manifest.xml in this folder
5. Click Upload

REQUIREMENTS
------------
- Microsoft Excel 2016 or later (desktop or web)
- Internet connection (to load the add-in from SharePoint)
- Must be logged into your Goodly Insights Microsoft 365 account
